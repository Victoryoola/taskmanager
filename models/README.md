# Task Manager API
An API built with express.js and MongoDB
This API allows users to perform CRUD operations such as create, update, delete with validation included

# Features
1. Create task
2. Retrieve All created tasks
3. Retrieve one single task
4. Update an existing task
5. Delete a task
6. Validation when creating a task

# Tech Used
1. Node.js
2. Express.js
3. MongoDB
4. Postman
